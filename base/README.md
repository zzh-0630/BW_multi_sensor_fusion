base层：放一些底层传感器驱动代码

- bw_ros2_driver：北微传感产品的ros2驱动，基于ubuntu22.04，ros2 humble。

- bw_ros_driver：北微传感产品的ros驱动，基于ubuntu20.04，ros1 noetic。

- livox_ros_driver2：览沃激光雷达ros2驱动，测试雷达为MID360，基于ubuntu22.04，ros2 humble。

- OrbbecSDK_ROS2：奥比中光ros2驱动，测试相机为gemini 335L，基于ubuntu22.04，ros2 humble。

- usb_cam：V4L类型相机的ros2驱动程序，基于ubuntu22.04，ros2 humble，可切换分支调整适配版本。
